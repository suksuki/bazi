## 🚨 关键问题诊断

### 问题 1：C07 事业相严重恶化

**V20.0：** 模型=70.7, GT=80.0, MAE=9.3
**V21.0：** 模型=51.3, GT=80.0, MAE=28.7

**诊断：**
- 模型得分从 70.7 下降到 51.3（下降 19.4 分）
- 可能原因：
  1. **月令权重从 1.2 提升到 2.0**：可能过度增强了月令的影响，导致其他因素被削弱
  2. **复杂交互逻辑**：可能过度修正了基础能量
  3. **能量流耦合**：可能过度削弱了日主能量（多重抵消）

### 问题 2：C01 情感相严重恶化

**V20.0：** MAE ≈ 1.0
**V21.0：** MAE = 35.0

**诊断：**
- 模型得分从接近 GT 变成严重偏离
- 可能原因：
  1. **参数变化过大**：四柱权重变化可能改变了能量分布
  2. **复杂交互逻辑**：可能错误地应用了某些交互修正

### 问题 3：C04 财富相恶化

**V20.0：** MAE = 1.0
**V21.0：** MAE = 8.4

**诊断：**
- 模型得分从接近 GT 变成偏离

1. **三合/三会倍率过大：**
   - 三合倍率 3.0，三会倍率 4.0
   - 如果案例中有三合/三会，能量会被大幅放大
   - 可能导致能量分布失衡

2. **能量流耦合可能过度：**
   - 顺生链放大：1.20^2 = 1.44（44% 放大）
   - 合力共振：如果 N=3，放大 = 1.30（30% 放大）
   - 多重抵消：可能过度削弱日主能量

3. **参数变化过大：**
   - 月令权重从 1.2 提升到 2.0（67% 提升）
   - 可能改变了所有案例的能量分布
