## Part 2: The "Feedback Anchor" Protocol

### Strategy A: OP Verification Loop
The system relies on a **Signal-to-Noise Filter**:
1.  **Identify OP**: Extract the UserID of the thread starter.
2.  **Scan for Interaction**: Ignore threads where OP post count == 1.
3.  **Keyword Sniffing**:
    - Positive: `["反馈", "准", "确实", "应期", "结婚", "升职"]`
    - Negative: `["不对", "反了", "错"]`
4.  **Action**: Only ingest threads with > 0 Keyword Hits from OP.
