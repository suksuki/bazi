### ✅ MOD_12 时空场惯性：衰减常数应用

**状态**: ✅ **已完成**

**实现文件**: `core/trinity/core/engines/spacetime_inertia_v13_7.py`

**核心物理问题**: 流年切换如何具有物理惯性？

**解决方案**:
- 指数衰减：W(t) = exp(-t / tau)
- 惯性常数：tau = 3.0 个月（默认值）
- 平滑过渡：确保能量变化不是"瞬断"的，而是有物理惯性的过渡

**核心公式**:
```
1. 衰减权重：W(t) = exp(-t / tau)
   其中：t 为时间（月），tau 为惯性常数（默认 3.0 个月）

2. 平滑能量：E_smooth = E_prev * W(t) + E_current * (1 - W(t))

3. 粘滞指数：Viscosity_Index = 1 - (min_energy / max_energy)
```

**核心功能**:
1. `calculate_inertia_weights()`: 计算惯性权重（指数衰减）
2. `apply_inertia_smoothing()`: 应用惯性平滑（流年切换平滑化）
3. `calculate_viscosity_index()`: 计算粘滞指数（场混合状态）
