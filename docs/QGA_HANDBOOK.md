## 核心定义

量子通用架构将格局视为**相干态 (Coherent State)**：

- **波函数坍缩**: 流年注入 = 观察者测量 → 系统从潜伏态向确定态坍缩
- **量子隧穿**: 通过"通关介质"(财/印)，能量可穿过克制壁垒
