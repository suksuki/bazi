## 💡 核心发现

### 误判模式分析

1. **Weak → Special_Strong (多个案例)**
   - 说明当前算法对"专旺格"的判定可能过于宽松
   - 或者这些案例的ground_truth标注有误

2. **Strong → Weak (多个案例)**
   - 说明Strong的判定可能过于严格
   - 或者这些案例的数据有问题

3. **Balanced → Special_Strong**
   - 说明Balanced和Special_Strong的边界需要调整
