# QGA 格局逆向审计规范 (RSS-V1)

## 一、核心哲学：逆向第一性原理

不同于传统的"查书找标签"，RSS-V1 遵循：**物理受力建模 -> 原局纯净海选 -> 动态场能仿真 -> 逻辑奇点提取 -> 古典/现代语义校验**。其核心逻辑是先通过算法定义物理本质，再通过大数据验证和修正古典经验。

## 二、动态注入模型：时空场能层级 (一命二运三风水)

系统的能量审计严格遵循物理权重层级与维度属性：

### 层级权重分配

- **一命（原局）**：基础能量场，权重 100%
- **二运（大运）**：静态场能修正，权重 50%（最高优先级基准场修正）
- **三风水（地理）**：全局阻尼系数，权重 ±15%（控制在 ±15% 以内）
- **流年**：关键瞬时触发脉冲，权重 30%（高频脉冲变量）

## 三、审计执行标准步骤 (Standard Procedures)

### Step A: 物理模型预设与原局海选

- **模型构建**：在 `registry.json` 中定义格局的物理公理（主导向量、应力阈值、相位角）。
- **纯净海选**：海选阶段只考虑原局特征。在 51.84 万静态样本库中，剥离所有动态干扰，锁定最符合公理的"格局母体"。
- **样本分型**：将海选结果分类为"稳态母体"与"缺陷母体（潜在奇点）"。

### Step B: 动态因子层级注入仿真

1. **大运静压**：注入大运场能，观察原局晶格的能级位移与变形情况。
2. **流年引动**：注入高频脉冲变量，测试系统在何时触发 `Logic_Collapse`（逻辑坍缩）。
3. **地理调优**：注入地理阻尼系数，验证环境对最终稳定性结果的修正力度。

### Step C: 语义对撞与奇点提炼

- **画像坍缩**：LLM 结合 `MatrixRouter` 输出动态受力后的 `energy_state_report`。
- **判词对比**：将物理因果画像与古典判词对撞，识别"古今语义差异"。
- **奇点标注**：标注稳定性 < 0.15 但古典描述缺失的"新格局"样本。

### Step D: 模型迭代与注册

- **算法调优**：若大运/流年的实际表现与物理预测不符，则修正场能渗透系数。
- **版本注册**：更新 `registry.json`，将经过验证的格局参数同步至 V25.x 注册表。

## 四、物理算子应用规范

### 能量叠加公式

$$E_{total} = [ (E_{base} \otimes \omega_{luck}) \oplus \Delta E_{year} ] \times (1 \pm \delta_{geo})$$

其中：
- $E_{base}$：原局基础能量
- $\omega_{luck}$：大运权重（锁定为最高优先级基准场修正）
- $\Delta E_{year}$：流年能量增量（关键瞬时触发脉冲）
- $\delta_{geo}$：地理修正系数（全局阻尼系数，控制在 ±15% 以内）

### 初始化参数

- **大运权重**：锁定为最高优先级基准场修正。
- **流年定位**：关键瞬时触发脉冲。
- **地理修正**：全局阻尼系数（控制在 ±15% 以内）。
