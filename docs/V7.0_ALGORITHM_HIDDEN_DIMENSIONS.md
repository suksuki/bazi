# V7.0 核心算法设计文档：隐藏维度 (Hidden Dimensions)

## 1. 核心定义：命运的隐藏维度

### 1.1 物理模型：场论与波叠加
我们参照之前的“墓库理论”和“三刑协议”，将这两个隐藏属性定义为 **“场论” (Field Theory)** 和 **“波叠加” (Wave Superposition)** 的问题。
要将“地域”和“国运”纳入算法，我们不能只把它们当作简单的加分项，而必须建立 **物理模型**。

## 2. 第一章：地域性 (Geography) - 空间能量场

### 2.1 命理基础描述 (Metaphysical Basis)
*   **概念**：“得地”与“失地”。《滴天髓》云：“东南西北，分野不同，五行强弱，随地而异。”
*   **物理本质**：**空间非均匀能量分布 (Spatially Inhomogeneous Energy Distribution)**。
    *   地球不是一个均质的能量体。受地磁场、日照角度（纬度）、洋流和地形影响，不同坐标点的 **基础五行背景辐射 (Background Radiation)** 是不同的。
    *   八字是时间的切片，而地域是 **空间的容器**。容器的物理属性（温度、湿度、磁偏角）直接决定了化学反应（命运演化）的速率和方向。

### 2.2 算法模型描述 (Algorithm Model: The GeoEngine)
我们建立 **`Geo_Modifier` (空间修正系数)** 模型。它不是产生新事件，而是 **扭曲 (Distort)** 现有的能量强弱。

*   **模型公式**：
    $$ E_{real} = E_{bazi} \times (1 + \Delta_{geo}) $$

    *   $E_{real}$: 修正后的实际能量
    *   $E_{bazi}$: 原局计算出的能量（旺衰）
    *   $\Delta_{geo}$: 地域修正值 (Vector)

*   **计算逻辑**：
    1.  **五行矢量映射 (Mapping)**：
        *   **北方 (Water Vector)**: 纬度 > 40°N。水气增强，火气衰减。
        *   **南方 (Fire Vector)**: 纬度 < 25°N。火气增强，水气衰减。
        *   **沿海 (Wood/Dampness)**: 湿度高，生木，晦火。
        *   **内陆 (Metal/Dryness)**: 燥土生金。
    2.  **调候修正 (Climate Adjustment)**：
        *   **判定**：若 $E_{bazi}$ 为“过于寒湿”（如生于亥子月），且 Location 为“南方”。
        *   **计算**：触发 **"Thermal Compensation" (热补偿)**，全盘能量活性提升 20% ($\Delta_{geo} = +0.2$)。
        *   **反之**：若命局火炎土燥，且居南方，触发 **"Overheating" (过热)**，系统熵增，稳定性下降 ($\Delta_{geo} = -0.2$)。

## 3. 第二章：国运 (National Fate) - 宏观载波干扰

### 3.1 命理基础描述 (Metaphysical Basis)
*   **概念**：“天时”与“气运”。《皇极经世》提出的 **三元九运** 体系。个人命运是小周期，国运是大周期。
*   **物理本质**：**宏观载波与共振干涉 (Macro-Carrier Wave & Resonance Interference)**。
    *   国运是一条巨大的 **载波 (Carrier Wave)**，个人的运势是调制在上面的 **信号波 (Signal Wave)**。
    *   当个人频率与载波频率一致时，发生 **建设性干涉 (Constructive Interference)**，振幅叠加（顺势而为）。
    *   当个人频率与载波频率相反时，发生 **破坏性干涉 (Destructive Interference)**，振幅抵消（逆水行舟）。

### 3.2 算法模型描述 (Algorithm Model: The MacroEngine)
我们建立 **`Macro_Beta` (宏观贝塔系数)** 模型。它决定了分数的“天花板”和“地板”。

*   **模型公式**：
    $$ S_{final} = S_{geo} + \beta_{macro} $$

*   **计算逻辑**：
    1.  **定义时代主频 (Era Frequency)**：
        *   **Period 9 (2024-2043)**: 离火运 (Element = Fire)。特征：虚火、科技、离散、精神。
    2.  **风口匹配算法 (Trend Resonance)**：
        *   **Input**: 用户八字的“财星” (Wealth Element) 和“官星” (Power Element)。
        *   **Logic**:
            *   若 `User_Wealth_Element` == `Era_Element` (e.g., 喜火人的财星是火):
                *   判定为 **"Riding the Wave" (冲浪模式)**。
                *   算法：`Macro_Bonus = +15.0` (时代红利)。
            *   若 `User_DayMaster` == `Era_Element` (e.g., 火命人):
                *   判定为 **"Peer Competition" (比劫夺财)**。时代充满了你的竞争对手。
                *   算法：`Macro_Penalty = -5.0` (内卷系数)。
    3.  **经济周期叠加**：
        *   引入外部参数 `GDP_Growth_Rate` (模拟值)。
        *   当处于衰退期，所有人的 `Base_Score` 统一向下平移 (Shift Down)。

*   **地域性** 是 **L1 基础层** 的 **乘法因子** (Multiplier)，它改变了种子的生存环境（旺衰）。
*   **国运** 是 **L5 全局层** 的 **加法因子** (Additive Term)，它提供了时代的额外动能（红利）。
