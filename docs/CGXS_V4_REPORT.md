## 1. 物理模型：双级磁场耦合放大器 (SEGS)

在 QGA V4.0 中，财官相生被重新定义为一个自激循环系统：
- **官星 (Officer)**: 磁场约束力。决定系统承载上限。
- **财星 (Wealth)**: 泵入电流。提供系统动力。
- **SAI 公式**: $SAI = (E_{Wealth} / E_{Officer}) \times (1 - Stability)$
- **Stability**: 由大运地支的十二长生能级决定。

## 4. 专题注入算法算子 (V4.0 特供)

- **Officer_Stabilization (0.70)**: `壬申` 大运。
    - 戊/己土 (官星为木): 申为绝/胎位 (Stability 0.3/0.8) -> 高危。
    - 庚/辛金 (官星为火): 申为病位 (Stability 0.8) -> 中稳。
    - 丙/丁火 (官星为水): 申为长生位 (Stability 1.5) -> 极稳。
- **Wealth_Pump_Trigger (0.25)**: `乙巳` 流年互动 `壬申` 大运。
    - `巳申` 六合激活水能泵入。
