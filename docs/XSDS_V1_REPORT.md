## 非线性物理核心原理

### 1. 能量级数的"指数压制"

```
SAI = (Energy²) / Protection
```

偏印能量增加 2x → SAI 增加 4x (指数级)

### 2. 干支"相干性耦合"

地支合化不是简单加能量，而是改变天干对冲的**波函数**。

| 机制 | 效果 |
|------|------|
| 巳申合 | 改变火水对冲路径 |
| 地支冲 | 激活天干隐性压力 |

### 3. 距离与屏蔽的"反平方律"

```
FieldStrength ∝ 1 / Distance²
```

年柱 → 日主 衰减远大于 月柱 → 日主
