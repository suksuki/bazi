# Antigravity L2 补充文档：时空相对论
**主题**: 多维场域叠加逻辑
**依赖**: `ALGORITHM_CONSTITUTION_v3.0.md`

### 1. 宏观场 (Macro-Field)
* **定义**: 时代的背景辐射 (Background Radiation)。
* **逻辑**: 个体命运受当前历元（如三元九运）的主气影响。顺应时代主气的五行获得加权，背离的受到抑制。

### 2. 中观场 (Meso-Field)
* **定义**: 地理环境对物理常数的修正。
* **逻辑**: 
    * **纬度**: 修正“寒暖”参数。高纬度增加水/金权重，低纬度增加火/土权重。
    * **半球**: 南北半球的季节（月令）定义相反。

### 3. 微观场 (Micro-Field)
* **定义**: 观察者的相对时空坐标。
* **逻辑**: 
    * **真太阳时**: 所有排盘计算必须基于太阳过顶时间，而非行政时区时间。
    * **时差校准**: 经度差异直接转化为时间偏移量。
