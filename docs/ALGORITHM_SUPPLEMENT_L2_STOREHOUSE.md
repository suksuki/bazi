# Antigravity L2 补充文档：墓库拓扑学
**主题**: 墓库状态定义的纯逻辑描述
**依赖**: `ALGORITHM_CONSTITUTION_v3.0.md`

### 1. 物理定义
地支中的土元素 (辰/戌/丑/未) 具有 **波粒二象性**，可坍缩为两种基本状态：
1.  **库 (Vault)**: 高能态。能量被高压约束，处于潜伏期。
2.  **墓 (Tomb)**: 低能态。能量微弱，接近寂灭。

### 2. 状态转化逻辑 (State Transition)

#### 2.1 闭库态 (Sealed State)
* **定义**: 在没有外部强相互作用（冲/刑）时，能量处于“封印”状态。
* **逻辑**: 此时藏干能量不能被天干直接利用，或者利用率极低（高阻抗）。

#### 2.2 隧穿态 (Tunneling / Open Vault)
* **触发条件**: 发生 **对撞 (Clash)** 或特定条件的 **刑 (Punishment)**。
* **物理现象**: 势垒被击穿，发生 **量子隧穿效应**。
* **逻辑**: 
    1. 忽略土元素的本气克制。
    2. 释放出巨大的积蓄能量（系数 > 1.0）。

#### 2.3 坍塌态 (Collapse)
* **触发条件**: 当基础能量不足（处于“墓”态）时遭遇对撞。
* **物理现象**: 容器破碎。
* **逻辑**: 不产生能量释放，反而对命局结构造成震荡伤害 (Negative Score)。
