## 判据

1. **月令帝旺**: 月支 = 日主羊刃
2. **七杀透干**: 天干有七杀
3. **无印**: 印星强度 < 0.5
4. **无食**: 食神强度 < 0.5
5. **比值门槛**: 0.5 ≤ ratio ≤ 2.0

---

## 能量公式

```
P_blade = 3.0 + (羊刃数-1)×1.5 + 比劫数×0.5
P_sha = Σ(七杀权重) [年0.5 月3.0 日1.0 时0.8]
SAI = P_blade × P_sha × DistFactor / 10 × GEO
```
