## SAI 公式

```
SAI = (Xiao² / Shi) / (1 + Wealth × 0.5)
Energy_Output = max(0, 1 - SAI/10) × 100%
```

## 发现

1. **断路率 ~20%** - 约 1/5 样本完全断路
2. **财星保护率 34%** - SAI 下降约 1/3
3. **全年危险** - 所有年份 SAI > 4.0
4. **2025 最危险** - SAI 4.96，断路率 21.6%

## 断路判定标准

| SAI | 状态 |
|-----|------|
| < 1.0 | 安全 |
| 1.0-3.0 | 临界 |
| 3.0-5.0 | 高危 |
| ≥ 5.0 | **CIRCUIT_BREAK** |
